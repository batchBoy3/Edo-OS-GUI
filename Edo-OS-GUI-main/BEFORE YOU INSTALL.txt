System requirments
-CMD NT ver 10.01 or later 
-1GB ram (recommended 4GB or higher) 
-No graphics card needed 
-CPU running at 1.00 GHZ or higher 
-Must have USB support 
-Resolultion must be 1152 x 864 or higher 
-Internet must be 56KB per sec or higher 
-Must have atleast 8 bit speaker support 
-Must have atleast 42MB of empty disk space. Recomendations: Low end: 128 MB  Mid End: 256MB  High End: 512MB 
It is recommended to make a seperate partition just for Edo OS GUI. Use the values above for your PC end.

BEFORE-HAND SETUP
============================================================================================
FOR WINDOWS 7,8,8.1,10
Open the system32 cmd.exe (Not the one supplied with Edo OS GUI) and right click the top bar.
Click "Properties".
Select "Options"
Uncheck the checkbox "QuickEdit Mode".
Done.
FOR WINDOWS 11
Open Terminal.
Right-click the top bar. NOT THE TAB.
Click "settings"
Change the default profile to "command prompt"
Change the default terminal application to "Windows Console Host."
Now click "Save"
Now go to "Rendering"
Uncheck the option "Use the new text renderer(atlas-engine)"
Quit "Terminal"
Goto search and type cmd.exe.
Right-click the top bar.
Click "Properties"
Click "Options"
Uncheck the checkbox "QuickEdit mode"
Done.

Also! Copy ALL OF THE FILES AND FOLDERS TO YOUR DISK!!! Then run setup.bat
